# Event Desinator (Line entry in history list)

| Shortcut | Purpose | Example |
| ---------|---------|---------|
| ! | Start a history substitution | ! |
| !n | Refer to command *n* in history | !5 |
| !-n | Refer to current command minus *n* | !-5 |
